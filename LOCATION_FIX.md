# ✅ إصلاح مشكلة عرض الأماكن بدون اللوكيشن

## ❌ المشكلة السابقة

1. **لا تظهر الأماكن إذا كان اللوكيشن مغلق**
   - التطبيق كان يطلب اللوكيشن تلقائياً عند فتح أي صفحة
   - إذا رفض المستخدم، لا تظهر الأماكن نهائياً
   - رسالة "لا توجد أماكن" حتى لو كان هناك بيانات

2. **طلب اللوكيشن المتكرر**
   - كل مرة تدخل صفحة جديدة، يطلب اللوكيشن
   - مزعج للمستخدم

3. **عدم وضوح خيار "الأقرب"**
   - لو اللوكيشن مغلق وضغط "الأقرب"، لا يحدث شيء

---

### 1. عرض الأماكن بدون اللوكيشن

```dart
@override
void initState() {
  super.initState();
  // Load data first without location ✅
  _loadGyms();  // أو _loadPharmacies() أو _loadClinics()

  // لا يطلب اللوكيشن تلقائياً ❌
}
```

**النتيجة:** الأماكن تظهر مرتبة بالاسم حتى لو اللوكيشن مغلق ✅

### 2. طلب اللوكيشن عند الحاجة فقط

```dart
void _changeSortOption(String sortOption) {
  // If sorting by distance but location not available, request it
  if (sortOption == 'distance' && _userLocation == null) {
    _requestLocation();  // يطلب اللوكيشن الآن فقط
    return;
  }
  
  // باقي الكود...
}
```

**النتيجة:** يطلب اللوكيشن فقط عند الضغط على "الأقرب" ✅

5. المستخدم يوافق
6. ✅ الأماكن تترتب حسب الأقرب

```dart
        // ترتيب بالمسافة
        gyms.sort((a, b) => distA.compareTo(distB));
      } else {
        // fallback: ترتيب بالاسم
        gyms.sort((a, b) => a.name.compareTo(b.name));
      }
      break;
    // ...
  }
}
```
