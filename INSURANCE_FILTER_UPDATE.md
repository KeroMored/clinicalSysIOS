# تحديث واجهة قائمة الصيدليات - فلتر شركات التأمين

## التاريخ: 4 فبراير 2026

## نظرة عامة
تم تحديث واجهة قائمة الصيدليات لتحسين تجربة المستخدم من خلال:
1. إضافة بطاقة مرئية تشير إلى الصيدليات المتعاقدة مع شركات التأمين
2. إضافة فلتر سريع لعرض الصيدليات المتعاقدة مع التأمين فقط

## التحديثات المنفذة

### 1. بطاقة التأمين في PharmacyCard
**الملف:** `lib/features/pharmacy/presentation/widgets/pharmacy_card.dart`

#### التصميم الجديد:
تم إضافة بطاقة جميلة بلون **بني فاتح/لبني** تظهر تحت صف التقييم والقلوب مباشرةً.

#### المواصفات:
```dart
Container(
  // تدرج لوني بني فاتح
  decoration: BoxDecoration(
    gradient: LinearGradient(
      colors: [
        Color(0xFFFEF3E2),  // بيج فاتح
        Color(0xFFFDE7C8),  // بني فاتح
      ],
    ),
    borderRadius: BorderRadius.circular(8),
    border: Border.all(
      color: Color(0xFFD97706).withOpacity(0.3),
      width: 1,
    ),
  ),
  child: Row(
    children: [
      Icon(Icons.health_and_safety),  // أيقونة تأمين صحي
      Text('متعاقد مع شركات التأمين'),
    ],
  ),
)
```

#### شرط الظهور:
- يظهر فقط إذا `pharmacy.hasInsurance == true`
- و `pharmacy.insuranceCompanies.isNotEmpty`

#### الألوان المستخدمة:
- **الخلفية**: تدرج من #FEF3E2 إلى #FDE7C8 (بني فاتح/لبني)
- **الحدود**: #D97706 بشفافية 30%
- **الأيقونة**: #D97706 (برتقالي/بني)
- **النص**: #92400E (بني داكن)

### 2. فلتر شركات التأمين
**الملف:** `lib/features/pharmacy/presentation\screens\the_pharmacies_screen.dart`

#### الميزة الجديدة:
تم إضافة زر فلتر في AppBar لعرض الصيدليات المتعاقدة مع شركات التأمين فقط.

#### المتغير الجديد:
```dart
bool _filterInsuranceOnly = false;
```

#### الزر في AppBar:
```dart
IconButton(
  icon: Icon(
    _filterInsuranceOnly 
        ? Icons.health_and_safety          // ممتلئ عند التفعيل
        : Icons.health_and_safety_outlined, // فارغ عند عدم التفعيل
    color: _filterInsuranceOnly 
        ? Color(0xFFD97706)  // برتقالي/بني عند التفعيل
        : Color(0xFF64748B), // رمادي عند عدم التفعيل
  ),
  tooltip: _filterInsuranceOnly 
      ? 'إظهار الكل' 
      : 'شركات التأمين فقط',
  onPressed: () {
    setState(() {
      _filterInsuranceOnly = !_filterInsuranceOnly;
      _pharmacies.clear();
      _lastDocument = null;
      _hasMore = true;
    });
    _loadPharmacies();
  },
)
```

#### الفلترة في Firestore:
```dart
Query query = FirebaseFirestore.instance
    .collection('pharmacies')
    .where('status', isEqualTo: 'approved');

// تطبيق فلتر التأمين إذا كان مفعلاً
if (_filterInsuranceOnly) {
  query = query.where('hasInsurance', isEqualTo: true);
}
```

## كيفية الاستخدام

### للمستخدمين:

#### 1. رؤية الصيدليات المتعاقدة:
- في قائمة الصيدليات، ابحث عن البطاقة البنية الفاتحة
- تظهر تحت صف التقييم والقلوب مباشرةً
- تشير إلى أن الصيدلية متعاقدة مع شركات تأمين

#### 2. فلترة الصيدليات:
1. في شاشة "جميع الصيدليات"
2. اضغط على أيقونة التأمين الصحي في الأعلى (بجانب أيقونة الترتيب)
3. ستتغير الأيقونة إلى اللون البرتقالي/البني
4. سيتم عرض الصيدليات المتعاقدة مع التأمين فقط
5. للعودة لعرض جميع الصيدليات، اضغط على الأيقونة مرة أخرى

## الفوائد

### 1. تحسين تجربة المستخدم:
- رؤية سريعة للصيدليات المتعاقدة مع التأمين
- لون مميز (بني فاتح) يسهل التعرف عليه
- فلترة سريعة بضغطة واحدة

### 2. وضوح المعلومات:
- معلومة واضحة ومباشرة
- تصميم جذاب ومتناسق مع باقي الواجهة
- أيقونة معبرة (health_and_safety)

### 3. سهولة الوصول:
- الفلتر في مكان بارز (AppBar)
- تغيير الحالة بصري واضح (لون + شكل الأيقونة)
- Tooltip يشرح الوظيفة

## ملاحظات تقنية

### 1. الأداء:
- الفلترة تتم على مستوى Firestore (server-side)
- تحميل كسول (Pagination) يعمل مع الفلتر
- إعادة تحميل البيانات عند تغيير الفلتر

### 2. Index في Firestore:
قد تحتاج إلى إنشاء مركب في Firestore Console:
```
Collection: pharmacies
Fields: status (Ascending), hasInsurance (Ascending), name (Ascending)
```

لإنشاء الـ Index:
1. افتح Firebase Console
2. اذهب إلى Firestore Database
3. اختر Indexes
4. أضف Index مركب جديد
5. أضف الحقول المذكورة أعلاه

### 3. الحالات الخاصة:
- إذا لم تكن هناك صيدليات متعاقدة، سيظهر "لا توجد صيدليات"
- الفلتر يعمل مع البحث والترتيب
- يمكن استخدام الفلتر مع "الأقرب" و"الأعلى تقييماً"
